Projet

Un PPE (Projet Personnalisé Encadré) est un projet informatique qui peut être réalisé seul ou par groupe de 2 à 5 personnes. Pour cette année, nous devons préparer 3 PPEs autour du club Lyon Palme :

- Un client léger :

Notre mission ici est de créer une application intégrée (développée en Python avec le Framework Django) au site de Lyon Palme permettant de créer le planning des entrainements. Une application utilisé principalement par les entraineurs. Elle permettra l’affectation des entraineurs aux entrainement en piscine pour les participants.

- Un client lourd :

L'application Client lourd a pour but la gestion de matériel du club Lyon Palme. Pour cette application, on va donc utiliser comme langage le C# (sur Visual Studio) et le langage SQL (pour la création des tables).
